## Recommended Typography

### Primary recommended combination
- **Heading:** Canela
- **Body:** Montserrat Light
- Vibe: feminine, elegant, timeless.
- Note: use Canela only if you have the correct web license. For a free fallback, use **Cormorant Garamond** or **Playfair Display**.

### Alternative premium/editorial combination
- **Heading:** Saol Display
- **Body:** Neue Haas Grotesk Light
- Vibe: artistic, editorial, luxury.
- Free fallback: **Cormorant Garamond** + **Inter Light**.

### Alternative minimal/sophisticated combination
- **Heading:** Editorial New
- **Body:** Inter Light
- Vibe: minimal, modern, sophisticated.
- Free fallback: **DM Serif Display** + **Inter**.
